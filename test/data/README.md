# test/data

Public RoundAbout test fixtures.

This directory contains the small public fixture set used by the generic RoundAbout tests in this repository.

Typical contents include:

- Solidity source files
- Certora-style `.conf` files
- Optional spec files
- Pre-generated AST artifacts in `ast/`
